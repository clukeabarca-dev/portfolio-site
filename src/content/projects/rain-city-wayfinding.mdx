## Brief

Rain City Wayfinding explores how a location-based design system can feel useful before it feels branded. The project imagines a set of public-facing touchpoints for newcomers, locals, and visitors moving between neighborhoods.

## Concept

The design language is based on movement: ferry lines, transit signs, hill grades, rain marks, and the way Seattle alternates between dense street grids and open water.

## Design System

The system uses sturdy sans-serif labels, compact maps, and soft contrast. Color does not try to overpower the environment. It sits inside it.

## Outcome

The project page should behave like a route: a large opening image, then chaptered sections that move from orientation to detail.
