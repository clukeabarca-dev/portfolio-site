## Brief

Create a poster and fine-art print series that can live comfortably in home interiors while still feeling graphic and intentional.

## Concept

The work explores abstract composition, soft materiality, and controlled color to create pieces that feel quiet but memorable.

## System

The project can include print editions, poster mockups, framing studies, colorways, and production notes.

## Outcome

This case study should show the print work as both a designed system and a set of standalone visual objects.
