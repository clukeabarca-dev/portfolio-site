## Brief

Archive Objects is the underlying content pattern for the portfolio itself. It treats each project as a set of designed artifacts with context, not just a gallery of final images.

## Concept

The visual language borrows from exhibition indexes, product specification sheets, and editorial contents pages. The result is a portfolio format that can hold both polished outcomes and process fragments.

## Design System

Each object includes a title, medium, year, role, and a short contextual note. This gives the archive a consistent rhythm while leaving room for each project to carry its own visual tone.

## Outcome

This system becomes the practical bridge between the index-like archive and the cinematic project case studies.
